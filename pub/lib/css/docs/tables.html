Table borders customization

Mixin .table-bordered() provides table borders customizations. To setup table borders use following allowed values to @_type variable:

  normal - all borders are set up
  horizontal - table head and table cells have only horizontal borders
  vertical - table head and table cells have only vertical borders
  light - only table head cells have bottom border
  clear - table does not have borders
.example-table-6 {
    .table();
    .table-bordered();
}

Table with horizontal borders

To set only horizontal borders to table you should set variable:

  .table-bordered(@_type: horizontal);
.example-table-7 {
    .table();
    .table-bordered(
        @_type: horizontal
    );
}

Table with vertical borders

To set only horizontal borders to table you should set variable:

  .table-bordered(@_type: vertical);
.example-table-8 {
    .table();
    .table-bordered(
        @_type: vertical
    );
}

Table with light borders

To set bottom borders only to table caption you should set variable:

  .table-bordered(@_type: light);
.example-table-9 {
    .table();
    .table-bordered(
        @_type: light
    );
}

Table without borders

To set bottom borders only to table caption you should set variable:

  .table-bordered(@_type: clear);
.example-table-10 {
    .table();
    .table-bordered(
        @_type: clear
    );
}

Responsive table technics #1

MageLess framework employs two responsive tables technics which are applyed by calling appropriate mixin. .table-overflow() mixin call in media queries adds a horizontal scrollbar for mobile version. This mixin accepts no variables.

.example-table-13 {
    .table();
    .table-bordered();
    .table-striped();
    .table-hover();
}

@media only screen and (max-width : @break-point-1) {
    .example-table-14 {
        .table-overflow();
    }
}

Responsive table technics #2

.table-responsive() mixin call in media queries rearranges table cells for mobile version.

.example-table-15 {
    .table();
    .table-bordered();
    .table-striped();
    .table-hover();
}

@media only screen and (max-width : @break-point-1) {
    .example-table-15 {
        .table-responsive(
            @_table-bg-responsive: #fc0,
            @_table-th-bg-responsive: orange,
            @_reset-table-striped: true,
            @_reset-table-hover: true
        );
    }
}
